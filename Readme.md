# METTA. Верска сайт.
---

Проект представляет собой одностраничный сайт.

## Стек
* HTML
* CSS
* JS Vanilla Library integrate

## Реализация
* Flexbox вёрстка
* CSS-анимация
* Файловая структура проекта по правилам Nested БЭМ
* Адаптивный сайт
* Добавление анимаций JS с помощью сторонних библиотек
